# Auto-git

O auto-git é um projeto pessoal desenvolvido a fim de automatizar comandos em git, realizados por linha de código. O sistema conta com uma interface gráfica simples e objetiva, que permite ao usuário escolher entre as principais funções utilizadas no dia a dia do desenvolvedor:

* Git merge (permite a escolha entre as branchs do projeto)
* Git branch (permite a navegação entre as branchs)
* Git branch -d (deletar branchs do projeto)

Para utilizar o sistema, é nescessário ter instalado no seu computador o pacote Fzf (fuzzy finder).
  - repositório com tutorial de instalação do Fzf: https://github.com/junegunn/fzf

Com o Fzf instalado, baixe o script e abre o Bash na pasta onde ele está salvo, após isso, execute-o via bash

```./auto-git-script.sh```
